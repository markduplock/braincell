# Braincell

## A tiny personal wiki for my tiny server

Braincell is a tiny local wiki with big plans.

The goal is to learn HTTP, routing, forms, file storage, auth, databases, and deployment by building a small private wiki from scratch.

## Planned features

- View pages
- Create, edit, and delete pages
- Page index
- Basic search
- Markdown rendering
- Tags and categories
- SQLite storage
- HTML/CSS frontend
- Users, roles, and simple auth
